# 0x05-processes_and_signals

This directory contains Bash scripts that demonstrate process and signal handling in Linux.

## Scripts

- 0-what-is-my-pid: Prints the PID of the running script.

## How to run

1. Make the script executable:

chmod +x script_name

2. Run the script:

./script_name

Replace script_name with the actual script you want to run.
